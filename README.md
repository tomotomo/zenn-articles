## How to write

### 新しい記事の作成

```zsh
npx zenn new:article --slug 記事のスラッグ --title タイトル --type idea --emoji ✨
```

Publicationの新しい記事の作成

```zsh
npx zenn new:article --slug 記事のスラッグ --title タイトル --type idea --emoji ✨
```

### プレビュー

```zsh
npx zenn preview
```
